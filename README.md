# Reserveerimissüsteem

Tehnoloogia: Spring boot backend, postgres andmebaas ja Vue.js frontend.

## Töö jaotus ja kulunud aeg

**Algus ja ettevalmistus 2h**

- Lõin projekti struktuur ning esialgsed objektid.
- Siin võttis peamiselt aega uurimine ja varasematest projektidest jdbc meeldetuletamine.

**Äriloogika ja soovitussüsteem 4h**

- Lõin controllerid ja soovitus _Service_-i, ja lõpetasin ülejäänud backend osa.

**Front end disain 8h**

- Kasutasin taustapildina restorani plaani
- Kujutasin lauad üle selle tausta, kasutades protsentuaalseid asukohti, mida salvestasin andmebaasis
- Lisaks parandasin soovitusalgoritmi, et kui puudub 1 laud kuhu kogu seltskond ära mahuks soovitataks kaks lähestikust lauda
- Lisasin päevaprae _navbari_

See osa mu tööst võttis kõige kauem aega, kuna olen sellega ka kõige vähem kokku puutunud, ma konsulteerisin oma kursusematerjale.



## Probleemid ja lahendused

Kuna ma ei ole Springbootiga väga palju kokku puutunud siis vaatasin palju Youtube'i videosid ja kursuseid selle kohta.
